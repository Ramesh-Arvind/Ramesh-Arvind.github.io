# ramesh-arvind.github.io

Personal academic website of Ramesh Arvind Naagarajan.
Built with Jekyll, deployed via GitHub Pages.

## Local preview

```bash
bundle install
bundle exec jekyll serve
```

Visit http://localhost:4000.

## Adding a blog post

Create a new file in `_posts/` named `YYYY-MM-DD-slug.md` with the front
matter you can copy from the existing post.

## Updating the CV

Edit `pages/cv.md` for the web version. Replace
`assets/pdf/ramesh-arvind-cv.pdf` for the downloadable version.
